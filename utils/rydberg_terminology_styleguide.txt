里德堡量子计算/量子模拟实验术语规范指南（中英对照）
版本：v1.0
来源：chapters/chapter_01..04（含各章 Decision_Records.md）与 source/en 全部 .tex

1. 总则（书写与缩写）
1) 首次出现格式：中文术语（英文全称, 缩写）或 中文术语（英文全称）
   - 例：光镊（Optical Tweezers），空间光调制器（Spatial Light Modulator, SLM）
   - 例：反时序关联函数（out-of-time-ordered correlator, OTOC）
2) 后续出现：优先使用中文术语；缩写在需要强调/与图中标注一致时保留。
3) 大小写：专名/缩写保持英文原大小写（Rydberg, MOT, SLM, AOD, PDH, ULE, OTOC, PXP）。
4) 连字符：英文原文为 collapse-and-revival 时，统一保持连字符；中文统一“塌缩与复苏”。
5) “寻址/定址”：全文统一使用“寻址”作为 addressing 的译法。
6) “失谐/失调”：全文统一使用“失谐”作为 detuning 的译法。

2. 核心平台与概念术语（必须统一）
- Optical Tweezers / Optical Tweezer -> 光镊
- Optical Dipole Trap -> 偶极阱
- Rearrangement / Sorting -> 原子重排
- Atom Assembly -> 原子组装（仅用于算法/流程名称；不要与“原子重排”混用）
- Rydberg Blockade -> 里德堡阻塞（必要时写“里德堡阻塞效应”）
- Neutral Atom -> 中性原子
- Qubit -> 量子比特
- Single Atom Detection -> 单原子成像（或“单原子分辨探测”；同一章节内二选一并保持一致）
- Site-resolved / Single-site -> 单址分辨 / 单址（建议用“单址分辨”作为正式表述）
- Addressing (laser/beam) -> 寻址（激光/光束）

8. 符号与定义（来自术语/符号约定）
- 拉比频率：使用 Ω（不使用 ω_R）
- 失谐量：Δ = ω_laser - ω_atom（蓝失谐为正，红失谐为负）
- 主量子数：n（里德堡态）；有效主量子数：n*
- 超精细量子数：F；磁量子数：m_F

9. 术语一致性检查清单（写作时自检）
1) 同一章内不要混用：寻址/定址、失谐/失调、禁闭-解禁闭/禁闭-去禁闭、重排/组装。
2) 首次出现是否给出英文全称与缩写（如 SLM/AOD/EOM/PDH/OTOC/EIT）？
3) “Holevo 信息”“OTOC”等是否保持统一写法（不音译、不换译）？
4) 关键平台词是否命中统一译法：光镊、偶极阱、原子重排、里德堡阻塞、单原子成像。
