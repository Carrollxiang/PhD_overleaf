# 实现单比特门与保真度标定的技术报告

## 一、 单比特门技术简介

在中性原子量子计算系统中，实现高保真度的单比特门操作是构建可扩展量子计算机的重要基础。中性原子阵列平台上，我们通常使用拉比振荡（Rabi oscillation）来实现单比特量子门操作。这种方法利用激光（微波）与原子的相干相互作用，通过调控光场（微波场）的频率、相位和持续时间，可以实现量子态在基态的相干演化。然而，在实际操作中激光（微波）的相位噪声、频率漂移以及原子的运动会导致单比特门保真度的降低。因此为了获得高保真度的单比特门操作，我们需要精确控制光场（微波场）的参数，并且尽可能减少环境噪声的影响。

本技术报告将详细介绍我们在中性原子阵列平台上实现单比特门的实验过程，包括拉比振荡的观测与表征、单比特门操作的实现方法，以及通过 randomized benchmarking 测量得到的门保真度结果。我们还将讨论影响门保真度的主要因素，并提出相应的优化方案。

## 二、 单比特门技术方案

### 2.1 单比特门复合脉冲序列

在量子计算系统中，系统误差往往是影响量子操控精度的主要来源。相比随机误差，系统误差更加难以通过简单的校准来消除，尤其是在控制大规模量子系统时，或者当所需的控制精度远超出测量精度时。幸运的是，在不需要知道具体误差大小的情况下，我们可以通过复合脉冲序列（composite pulse sequence）的方法来补偿系统误差。这种方法通过将单个不完美的脉冲替换为特定设计的脉冲序列，利用对称性使误差相互抵消。在核磁共振领域，这种方法已经得到了广泛应用。近年来，随着量子计算对门操作保真度要求的不断提高，复合脉冲序列技术在离子阱和中性原子阵列等量子计算平台上也展现出重要价值。通过精心设计的复合脉冲序列，我们可以将系统误差从 $O(\varepsilon)$ 降低到 $O(\varepsilon^n)$，（$n$ 为任意正整数），这为实现高保真度的量子门操作提供了有力工具。下面介绍如何针对不同误差补偿的复合脉冲序列。

首先确定，所有构成复合脉冲序列的脉冲，都是与操控的两个原子能级谱线共振的，也就是说单次脉冲的效果可以通过以下旋转算符表示：

$$R_{\phi}(\theta) = \exp \left[ -\frac{i\theta}{2} \sigma_{\phi} \right],$$

其中 $\theta$ 为单次旋转的角度，$\sigma_{\phi} = \sigma_x \cos \phi + \sigma_y \sin \phi$ 用于描述旋转轴，$\phi$ 为旋转轴和 Bloch 球的 x 轴的夹角。这样的好处就是省去了定义能级失谐的自由度，毕竟在实验上实现共振比精准调节失谐要更加简单。

### 2.1.1 针对功率抖动的补偿脉冲

根据【Phys. Rev A 70, 052318 (2004)】所述，实际操作中，旋转角度 $\theta$ 会因为功率抖动等原因，变成 $\theta(1 + \epsilon)$，因此一次真实的旋转操作可以表示为 $M_{\phi}(\theta) = R_{\phi}(\theta(1 + \epsilon))$，并且 $\epsilon$ 是在实验中未知量，不过假设在单次序列操作中，$\epsilon$ 恒定不变，这里只讨论两次实验间的功率抖动。为了补偿功率抖动，我们希望构建一组脉冲 $R_{\phi}^{[n]}(\theta) = \prod_i M_{\phi_i}(\theta_i)$，实现 $R_{\phi}^{[n]}(\theta) = R_{\phi}(\theta) + O(\epsilon^{n+1})$。

引入两条引理

$$R_{\phi}(-\theta\epsilon) M_{\phi}(\theta) = R_{\phi}(\theta)$$

$$M_{\phi}(2k\pi) = R_{\phi}(2k\pi(1 + \epsilon)) = \pm R_{\phi}(2k\pi\epsilon), \quad k = 0, 1, 2 \dots$$

我们的目标就是利用 $\prod_l M_{\phi_l}(2k_l\pi) = \pm \prod_l R_{\phi_l}(2k_l\pi\epsilon)$ 来构造 $R_{\phi}(-\theta\epsilon)$，这样就能实现 $(\prod_l M_{\phi_l}(2k_l\pi)) M_{\phi}(\theta) = R_{\phi}(\theta)$。根据 Trotter-Suzuki 公式，存在一个哈密顿 $B$ 和一组哈密顿 $\{A_l\}$ 满足 $B = \sum_l A_l$，那么就一定存在一组实数 $\{p_{jn}\}$ 满足

$$\exp(-iBt) = \prod_{j,l} \exp(-i p_{jn} A_l t) + O(t^{n+1}),$$

并且 $\sum_j p_{jn} = 1$。

不失一般性，先设定目标分解的算符为 $R_0(-\theta\epsilon) = \exp \left( i \frac{\theta\epsilon}{2} X \right) = \exp \left[ -i \left( -\frac{\theta}{2} X \right) \epsilon \right]$，令 $t = \epsilon, B = -(\theta/2)X$，选取 $A_1 = A_3 = m\pi(X \cos \phi + Y \sin \phi), A_2 = 2m\pi(X \cos \phi - Y \sin \phi)$，并且 $m, \phi$ 满足 $4m\pi \cos \phi = \theta/2$ ($A_1 + A_2 + A_3 = 4m\pi \cos \phi X = \frac{\theta}{2} X = \frac{\theta}{2} \sigma_0$)，以及 $q_{jn} = p_{jn} m$ 为整数。那么第 n 阶的纠正序列为

$$R_0(-\theta\epsilon) = \exp\left(i\frac{\theta\epsilon}{2}X\right) = \exp\left[-i\left(-\frac{\theta}{2}X\right)\epsilon\right] \exp(-iB\epsilon) = \Pi_{j,l} \exp(-i p_{jn} A_l \epsilon) + O(\epsilon^{n+1})$$

$$= \Pi_j \exp(-im\pi \sigma_\phi p_{jn} \epsilon) \exp(-i2m\pi \sigma_{-\phi} p_{jn} \epsilon) \exp(-im\pi \sigma_\phi p_{jn} \epsilon) + O(\epsilon^{n+1})$$

$$= \Pi_j \exp\left(-i \frac{2\pi q_{j,n}\epsilon}{2} \sigma_\phi\right) \exp\left(-i \frac{4\pi q_{j,n}\epsilon}{2} \sigma_{-\phi}\right) \exp\left(-i \frac{2\pi q_{j,n}\epsilon}{2} \sigma_\phi\right) + O(\epsilon^{n+1})$$

$$= \Pi_j R_\phi(2\pi q_{j,n}\epsilon) R_{-\phi}(4\pi q_{j,n}\epsilon) R_\phi(2\pi q_{j,n}\epsilon) + O(\epsilon^{n+1})$$

$$= \Pi_j M_\phi(2\pi q_{j,n}) M_{-\phi}(4\pi q_{j,n}) M_\phi(2\pi q_{j,n}) + O(\epsilon^{n+1}).$$

因此补正序列 $F_n = \Pi_j M_\phi(2\pi q_{j,n}) M_{-\phi}(4\pi q_{j,n}) M_\phi(2\pi q_{j,n}) = R_0(-\theta\epsilon) + O(\epsilon^{n+1})$。那么把补正序列加上之后

$$F_n M_0(\theta) = [R_0(-\theta\epsilon) + O(\epsilon^{n+1})] R_0(\theta) R_0(\theta\epsilon) = R_0(\theta) + O(\epsilon^{n+1}).$$

构建旋转操作

$$S_1(\phi_1, \phi_2, m) = M_{\phi_1}(m\pi) M_{\phi_2}(2m\pi) M_{\phi_1}(m\pi)$$

$$S_n(\phi_1, \phi_2, m) = S_{n-1}(\phi_1, \phi_2, m)^{4^{n-1}} S_{n-1}(\phi_1, \phi_2, 2m) S_{n-1}(\phi_1, \phi_2, m)^{4^{n-1}}$$

那么我们就可以构建复杂脉冲序列 $P_n$，递推公式满足：

$$P_0 = M_0(\theta),$$

$$P_2 = M_\phi(2\pi) M_{-\phi}(4\pi) M_\phi(2\pi) P_0,$$

$$P_{2j} = S_j(\phi_j, -\phi_j, 2) P_0,$$

其中 $\phi_j = \cos^{-1}(-\theta / 8\pi f_j); f_j = (2^{2j-1} - 2) f_{j-1}; f_1 = 1$。$P_{2j}$ 就是一组带通补偿脉冲，在较小的误差范围内，它会降低 infidelity，但是当误差变大的时候，它反而会放大误差。还可以用类似的方法构建宽带脉冲 $BB1 = S_1(\phi_{B1}, 3\phi_{B1}, 1) P_0$，其中 $\phi_{B1} = \cos^{-1}(-\theta / 4\pi)$。

为了验证参考文献【Phys. Rev A 70, 052318 (2004)】所提出的通带（passband）PB1 脉冲和宽带（broadband）BB1 脉冲对于补偿功率抖动的有效性，我们以 $R_0(\pi)$ 为例，比较了

### 2.1.2 针对失谐抖动的补偿脉冲

在功率抖动之外，还存在由于多普勒效应/外界电磁场等因素，造成的原子比特能级的抖动。因此也需要开发针对失谐抖动补偿的复合序列脉冲。

根据【[Phys. Rev. A 67, 042308 (2003)](https://doi.org/10.1103/PhysRevA.67.042308)】的内容，旋转操作 $R_\phi(\theta)$ 可以用三个脉冲进行等效替代

1、$(\cos \phi, \sin \phi, 0)$ 这个轴旋转 $\theta_1$
2、$(\cos(\phi + \pi), \sin(\phi + \pi), 0)$ 这个轴旋转 $\theta_2$
3、$(\cos \phi, \sin \phi, 0)$ 这个轴旋转 $\theta_3$

把这三个旋转矩阵相乘，然后只保留 $\delta / \Omega$ 的一次项，并与目标旋转矩阵进行比对，就得到了 $\theta_1, \theta_2, \theta_3$ 的值：

$$\theta_1 = 2n_1\pi + \frac{\theta}{2} - \arcsin \left( \frac{\sin \left( \frac{\theta}{2} \right)}{2} \right), \ n_1 = 0, 1, 2, \dots$$

$$\theta_2 = 2n_2\pi - 2\arcsin \left( \frac{\sin \left( \frac{\theta}{2} \right)}{2} \right), \ n_2 = 0, 1, 2, \dots$$

$$\theta_{3}=2 n_{3} \pi+\frac{\theta}{2}-\arcsin \left(\frac{\sin \left(\frac{\theta}{2}\right)}{2}\right), \quad n_{3}=0,1,2, \ldots$$

这组脉冲称为 CORPSE (compensation for off-resonance with a pulse sequence)。

我们以 $R_{0}(\pi)$ 为例，验证了 CORPSE 脉冲对于失谐抖动的补偿效果，以频率偏移 $error_{f}=\delta / \Omega$ 为横坐标，计算了 CORPSE 脉冲 infidelity 的变化曲线，如下图所示。

### 2.1.3 级联复合补偿脉冲

根据【[J. Phys. Soc. Jpn. 82, 014004 (2013)](https://doi.org/10.7566/JPSJ.82.014004)】，可以将上述的两种复合补偿脉冲进行级联，得到能同时补偿功率抖动和频率抖动的脉冲。比如将 BB1 脉冲的最后一段 $M_{\phi}(\theta)$ 替换成 CORPSE 脉冲，这样我们就得到了 reduced CinBB 脉冲：

$$CinBB(\theta, \phi)=M_{\phi_{B 1}}(\pi) M_{3 \phi_{B 1}}(2 \pi) M_{\phi_{B 1}}(\pi) M_{\phi}\left(\theta_{3}\right) M_{\phi+\pi}\left(\theta_{2}\right) M_{\phi}\left(\theta_{1}\right)$$

$$\theta_{2}=2 \pi-2 \arcsin \left(\frac{\sin \left(\frac{\theta}{2}\right)}{2}\right)$$

$$\theta_3 = \frac{\theta}{2} - \arcsin \left( \frac{\sin \left( \frac{\theta}{2} \right)}{2} \right)$$

## 2.2 钟态制备方案

为了实现钟态 $|5S_{1/2}, F=2, m_F=0\rangle$ 的态制备，我们先用两束 795 nm 激光进行最外态 $|5S_{1/2}, F=2, m_F=2\rangle$ 的态制备，然后用微波驱动跃迁 $|5S_{1/2}, F=2, m_F=2\rangle \rightarrow |5S_{1/2}, F=1, m_F=1\rangle$ 以及 $|5S_{1/2}, F=1, m_F=1\rangle \rightarrow |5S_{1/2}, F=2, m_F=0\rangle$。

用于最外态制备的 795 nm 激光由两个频率成分的激光组成：与 $|5S_{1/2}, F=1, m_F=1\rangle \rightarrow |5P_{1/2}, F=2, m_F=2\rangle$ 共振的 $\sigma^+$ 的 repumper，以及 $|5S_{1/2}, F=2, m_F=1\rangle \rightarrow |5P_{1/2}, F=2, m_F=2\rangle$ 共振的 $\sigma^+$ 的 depumper，两者共享同一个出光口，保证了相同的空间模式和偏振，指向与偏置磁场相同，helicity 为 $\sigma^+$。经过约 500 $\mu$s 的泵浦过程，所有原子被制备到最外态 $|5S_{1/2}, F=2, m_F=2\rangle$。

由于非钟态基态能级都是磁敏感态，并且微波的功率稳定性较高，以致这个过程的态制备主要误差来源为磁场抖动导致的基态能级抖动。因此，在最外态到钟态转移的微波操作中，我们采用 2 次 CORPSE 复合脉冲，来分别完成

$|5S_{1/2}, F=2, m_F=2\rangle \rightarrow |5S_{1/2}, F=1, m_F=1\rangle$ 与
$|5S_{1/2}, F=1, m_F=1\rangle \rightarrow |5S_{1/2}, F=2, m_F=0\rangle$ 的态转移过程，从而实现钟态态制备。

我们的微波系统配置如下：用微波源（BSVC20，BHE）提供 7.2 GHz 的射频信号，与 AWG 输出的 300~400 MHz 信号利用混频器混频。两个信号源均参考同一时钟信号，确保频率稳定。混频后，利用一个中心频率为 6.8 GHz、带宽为 200 MHz 的窄带滤波器，以确保仅有负一级频率边带（约 6.8 GHz）通过。滤波后的射频信号经过 20 W 微波放大器（HXPA7081G）进行功率放大。放大后的信号通过微波喇叭（LB-40400-KF，英联微波）发射至原子处，实现精确的态转移操作。该微波系统利用 AWG 的输出特性，可以任意编程输出频率、功率、时长和微波相位。而窄带滤波器的使用则有效避免了功率放大器中出现多频率信号导致的三阶互调失真及功率不稳定。

### 2.3 钟态相干操控方案

我们原子比特编码于钟态 $|0\rangle = |5S_{1/2}, F=1, m_F=0\rangle$ 和 $|1\rangle = |5S_{1/2}, F=2, m_F=0\rangle$。在实现了钟态 $|1\rangle = |5S_{1/2}, F=2, m_F=0\rangle$ 的制备之后，我们需要进行 $|0\rangle \leftrightarrow |1\rangle$ 的钟态能级相干操控，进而实现量子比特的控制。我们认为影响该比特相干性的主要因素为拉比频率的抖动，因此选用 BB1 复合脉冲来构建单比特门操作，以提高其保真度。

我们的主要论据如下：在 10 Gs 的偏置磁场下，两个钟态能级的一阶塞曼能级位移都是 0，二阶塞曼效应产生的能级差为 ~20 kHz。假设偏置磁场抖动为 1%，那么两个钟态比特的能级抖动 ~0.2 kHz。另一方面，在基态原子相干操作时，偶极阱的阱深 ~50 $\mu$K，对应基态的光位移 ~1 MHz，两个钟态能级的差分光位移 ~0.35 kHz，由于位置抖动导致的差分光位移抖动只会更小。因此，由于磁场、差分光位移抖动带来的能级抖动，相比于 ~10 kHz 的拉比频率，可以忽略不计，不太可能影响钟态能级的拉比振荡相干性。

# 三、 单比特门研究进展

### 3.1 钟态制备保真度的误差分析

由于我们将原子比特编码于钟态能级，钟态制备就成为了中性原子量子计算中的关键一环。因此，我们对钟态制备的保真度进行了误差分析。我们将钟态制备的 infidelity 主要分为如下 3 类：

(i) F=1 子能级的布居残留。这些布居残留在态探测中表现为亮布居（bright, B）。

(ii) F=2 非钟态子能级的布居残留。这些布居残留在态探测中表现为暗布居（dark, D）。

(iii) 原子损失。这些损失的布居在态探测中仍然表现为暗布居（dark, D）。

我们对于这些布居的测量基于态探测技术实现。态探测的基本原理为，首先利用吹飞（pushout）光对处于 F=2 子能级的原子进行破坏性清除，然后利用单原子探测技术来判断原子存在与否。通常我们认为吹飞光的清除效率超过 99.9%，其误差可忽略不计。因此，态探测的主要误差来源于原子探测误差，后者已在《原子探测保真度标定的技术报告》进行了标定。实验上，在钟态 $|1\rangle$ 制备完毕后，我们利用共振的微波操作，实现了 $|0\rangle \leftrightarrow |1\rangle$ 间的拉比振荡，如下图所示。通过拟合，拉比振荡中的最小布居为 1.9(4)%，最大布居为 98.5(4)%。说明钟态制备 infidelity 中，亮布居的贡献为 1.9(4)%，暗布居的贡献为 1.5(4)%。

亮布居 1.9(4)%的贡献，由态制备误差第(i)项、及正常 F=2 钟态子能级被误判为 B 的态探测误差所构成。根据《原子探测保真度标定的技术报告》的结论，态探测误差为 0.35(4)%，因此，态制备误差第(i)项的贡献为 1.9(4)% - 0.35(4)% = 1.6(4)%。

## 3.2 单比特门保真度的标定

在标定单比特门保真度的过程中，分析单次门操作测量结果，会遇到无法分离门操作误差、态制备误和测量误差的问题。因此在具体实验中，会通过执行长度为 L 的门来实现误差积累。首先将量子比特制备在钟态 $|1\rangle$，然后随机施加一系列单比特门操作，确保最终量子态能够转移到 $|0\rangle$ 态。在存在态制备与探测误差（SPAM errors）的情况下，末态中 $|0\rangle$ 态布居随 L 的关系满足 $P_0 = 1/2 + 1/2(1 - d_{if})(1 - d)^L$，其中 $d_{if}$ 表示 SPAM 误差，d 表示单个门的平均错误。通过拟合实验数据，可以提取出平均门保真度 F=1-d/2。根据所述方案，我们进行了单比特门保真度标定，结果如图 3.2 所示，通过拟合，我们可得**单比特门的保真度为 99.92(2)%**，满足单比特逻辑门保真度>99.9%的阶段二合同指标。
